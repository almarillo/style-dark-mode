# Vinted Dark Mode / Vinted Mode Nuit

A CSS style for vinted.fr usable in Stylus

Works on those domains :
- vinted.it
- vinted.nl
- vinted.fr
- vinted.co.uk
- vinted.es
- vinted.at

Works sitewide **except** :
- About page
- Our platform page


You can download Stylus on there : https://add0n.com/stylus.html
